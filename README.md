# NVIM Settings

This is a collection of NVIM repos and configs that I use.

## Dependencies

* fzf
```bash
git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf && ~/.fzf/install
```

* the_silver_searcher

```sudo apt-get install silversearcher-ag```

```brew install the_silver_searcher```
